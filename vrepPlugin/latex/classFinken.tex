\subsection{Detailed Description}
\hyperlink{classFinken}{Finken} class for handling any data exchanges between a F\+I\+Nken and Paparazzi. 

implementation of a \hyperlink{classFinken}{Finken} rotorcraft, includes communication loop.

Also handles the application of rotor mixing commands to the F\+I\+Nken.
